# 익명 커뮤니티 샘플

로그인/회원가입 후 간단한 익명 게시판을 사용할 수 있는 정적 데모입니다. 자유게시판·생활 TIP 게시판, 마이페이지, 오늘의 운세, 설정 영역이 포함됩니다.

## 주요 기능
- **로그인/회원가입**: 닉네임·이메일만으로 가입. 수집한 필수 정보는 XML 형태로 브라우저 저장소에 보관 예시 제공.
- **게시판**: 게시글 작성/수정/삭제(본인만), 첨부파일 이름 기록, 댓글 허용/비허용 선택, 게시판 필터.
- **마이페이지**: 내가 쓴 글만 갤러리 카드 형식으로 보기.
- **오늘의 운세**: 생년월일 기반 별자리·띠 계산, 오늘의 운세/2026년 신년 운세/토정비결 제공, PC 복사 / 모바일 공유 모달(Fallback 복사) 제공.
- **설정**: 로그아웃과 탈퇴(로컬 데이터 초기화).

## 실행 방법
정적 파일만으로 동작합니다. 단순히 로컬 서버를 띄워 접속하세요:

```bash
python -m http.server 8000
```

이후 브라우저에서 `http://localhost:8000`을 열면 됩니다. 모바일 웹을 확인하려면 개발자 도구의 디바이스 모드를 사용하세요.

## 데이터 보관 주의
실제 백엔드가 없는 데모이므로 모든 데이터는 브라우저 `localStorage`에 저장됩니다. 브라우저를 변경하거나 캐시를 삭제하면 데이터가 사라집니다.
